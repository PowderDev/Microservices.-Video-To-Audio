# Converter from video to audio

### What is used?
- Python
- Kubernetes
- RabbitMQ, MongoDB, MariaDB

### What is implemented?
- Simple Auth with JWT and a pre-added user
- Convertation from video to audio
- Notifications via email
- Simple web page
- Communication between services via RabbitMQ
